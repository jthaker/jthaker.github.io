Grapes of Math: Machine Learning with Wine


Chief Machine Learning Engineer. Utilized K-means clustering through scikit-learn to classify dataset on wine retrieved from www.ics.uci.edu. Variables chosen based on correlation matrix created by team member. Created linear regression model fitted to centroids of data clusters to more accurately determine and project relationship between chosen variables. Displayed fit curves and data using Seaborn and Matplotlib.

Project goal: To spot trends in wine data set. Group members used Tableau to display sales volume per country, popular varieties of wine, and indicate distribution of pricing per bottle. Additionally, machine learning through Python was used to extract clusters and find anomalies and outliers within data. Scikit-learn located centroids for clusters, which were then used to fit a linear regression to model a rough relationship between chemical qualities in wine.

Trends in Student Tuition and Enrollment


Python Developer. Initiated and assisted with data wrangling process using pandas. Created visuals for data using matplotlib. For statistical analysis, used numpy and scipy to fit data, analyze results and make predictions based on theoretical projections from regression models.

Project goal: To observe trends in student tuition, enrollment and aid available for students. To achieve this, we have looked at data for the following fields starting from 1973 all the way up to 2017: Tuition, Median Income, Student Aid, Wage Gap, Awarded Grants, Consumer Price Index, Student Aid Loans, and Enrollment Statistics.
